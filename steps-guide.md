# Step-by-Step Guide

## Step 1
Install Git on Linux.

## Step 2
Create a dedicated git user for better security.

## Step 3
Generate SSH keys for passwordless login.

## Step 4
Create a bare repository on server using:
git init --bare

## Step 5
Initialize local repository using:
git init

## Step 6
Commit changes and push to server.

## Step 7
Clone repository to verify setup.
